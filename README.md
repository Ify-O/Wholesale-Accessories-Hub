# Wholesale Accessories Hub

A responsive two-page website for Wholesale Accessories Hub built with HTML and CSS and deployed on Netlify. It includes a landing page, about section, and contact form.

This project serves as a scalable foundation, designed to expand features, improve UI, and integrate additional functionality over time.

## Project Structure

wholesale-accessories-hub/
│
├── index.html      (Home)
├── about.html      (About Us)
├── contact.html    (Contact Form)
├── style.css
└── script.js

## 🏠 Homepage

The homepage introduces the business with:
- Brand overview  
- Hero section  
- Store highlights  
- Success stories / reviews preview  
- Social media links  


## 👤 About Us

The About page includes:
- About the business  
- About the owner  
- Team section with images  
- Business mission and global reach  


## 📩 Contact

The Contact section includes:
- Netlify-powered contact form  
- Name, email, and message fields  
- Email notifications via Netlify Forms  
- Thank-you confirmation page  


## 🚀 Features

- Fully responsive design (mobile & desktop)  
- Clean multi-page structure  
- Netlify Forms integration (no backend required)  
- Social media integration (Instagram, TikTok, Facebook, Pinterest)  
- Scalable architecture for future expansion  


## 🌍 Deployment

Hosted on **Netlify** with GitHub integration for continuous deployment.
